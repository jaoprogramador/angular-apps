Es un proyecto para ver como crear la estructura de una aplicacion reactiva en Angular con un logueo que se autentica contra un API Rest que nos proporciona un token de acceso.

HAcemos un routing securizado de distintas páginas en las que podemos ver como pasar información entre componentes.

La aplicación hace uso de servicios y peticiones HTTP para el login o para obtener respuestas JSON y tratar la info por pantalla.

Se hace uso de PIPE, para definir componentes genéricos control de errores, validacion de formularios, componentes de Angular material...
